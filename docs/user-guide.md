# 员工使用手册

## 一、登录系统

### 1.1 Web 浏览器登录

1. 打开浏览器（推荐 Chrome、Edge、Safari）
2. 输入地址: `https://192.168.1.100`（具体地址问管理员）
3. 输入用户名和密码
4. 首次登录建议修改密码

### 1.2 手机 APP 登录

1. 下载 "Nextcloud" APP
   - iOS: App Store
   - Android: 应用商店

2. 打开 APP，点击"登录"
3. 输入服务器地址: `https://192.168.1.100`
4. 输入用户名和密码

## 二、文件管理

### 2.1 上传文件

**Web 端:**
1. 进入目标文件夹
2. 点击 "+" 按钮
3. 选择 "上传文件" 或 "上传文件夹"
4. 选择电脑上的文件

**手机端:**
1. 点击底部 "+" 按钮
2. 选择 "上传文件"
3. 可从相册、文件管理器选择

**批量上传图片（现场拍照）:**
1. 手机 APP → 设置 → 自动上传
2. 开启 "照片和视频"
3. 选择上传目标文件夹（如"施工现场"）
4. 拍照后自动同步到服务器

### 2.2 下载文件

**Web 端:**
- 单击文件可预览
- 点击 "..." → "下载" 可下载到电脑

**手机端:**
- 长按文件 → "下载"
- 文件保存到手机本地

### 2.3 在线预览

支持在线预览的文件类型:
- 📄 Word (.doc, .docx)
- 📊 Excel (.xls, .xlsx)
- 📽️ PowerPoint (.ppt, .pptx)
- 🖼️ 图片 (.jpg, .png, .gif)
- 📑 PDF (.pdf)
- 📝 文本 (.txt, .md)

**预览方式:**
- 直接点击文件名即可预览
- 图片可左右滑动浏览

### 2.4 在线编辑

**编辑 Office 文档:**
1. 点击文档（Word/Excel/PPT）
2. 点击"打开方式" → "ONLYOFFICE"
3. 在线编辑内容
4. 自动保存，关闭即保存

**协作编辑:**
- 多人可同时编辑同一文档
- 实时看到其他人的修改
- 右侧可添加评论

## 三、项目管理

### 3.1 项目文件夹结构

每个项目按标准模板创建：

```
2025-张三-阳光小区装修/
├── 01-设计阶段/
│   ├── 平面图/
│   ├── 效果图/
│   └── 设计方案.pdf
├── 02-合同文件/
│   ├── 装修合同.pdf
│   └── 报价单.xlsx
├── 03-材料清单/
│   ├── 主材清单.xlsx
│   └── 辅材清单.xlsx
├── 04-施工现场/
│   ├── 水电改造/
│   ├── 泥木工程/
│   ├── 油漆工程/
│   └── 竣工验收/
└── 05-验收文档/
    ├── 验收报告.pdf
    └── 保修卡.pdf
```

### 3.2 创建新项目

**方式 1 - 复制模板:**
1. 进入"01-进行中项目"
2. 右键点击"项目模板"
3. 选择"复制"
4. 粘贴并重命名为"2025-客户名-项目名"

**方式 2 - 新建文件夹:**
1. 进入"01-进行中项目"
2. 点击 "+" → "新建文件夹"
3. 按标准命名规则命名
4. 创建子文件夹结构

### 3.3 上传现场照片

**手机快捷上传:**
1. 打开 Nextcloud APP
2. 进入项目"04-施工现场"文件夹
3. 点击 "+" → "上传照片"
4. 选择照片 → 上传

**自动同步:**
1. 设置 → 自动上传
2. 选择"施工现场"作为目标文件夹
3. 拍照后自动上传

### 3.4 项目归档

项目完工后，需要归档：

1. 选中项目文件夹
2. 点击"移动或复制"
3. 选择"02-归档项目/2025年归档"
4. 点击"移动"
5. 联系管理员设置只读权限

**注意:**
- 归档后项目变为只读，不能修改
- 归档项目仍可查看和下载
- 需要修改请联系管理员解除归档

## 四、共享与协作

### 4.1 分享给同事

**内部共享:**
1. 选中文件/文件夹
2. 点击"分享"图标
3. 输入同事用户名
4. 选择权限（查看/编辑）
5. 点击"分享"

**部门共享:**
1. 选中文件夹
2. 点击"分享"
3. 选择用户组（如"项目经理"）
4. 整个组的人都可以访问

### 4.2 创建共享链接

**分享给客户（外部）:**
1. 选中文件/文件夹
2. 点击"分享"
3. 点击"创建共享链接"
4. 设置:
   - 🔒 密码保护（建议设置）
   - ⏰ 有效期（7天/30天/永久）
   - 👁️ 允许上传/编辑
5. 点击"复制链接"
6. 通过微信/邮件发送给客户

**客户使用:**
1. 点击链接
2. 输入密码（如有）
3. 查看或下载文件

### 4.3 添加评论

**在文件上评论:**
1. 打开文件（图片或文档）
2. 点击右侧"评论"图标
3. 输入评论内容
4. @某人 可通知对方

**查看评论:**
- 打开文件，右侧显示所有评论
- 点击评论可跳转到对应位置

## 五、版本历史

### 5.1 查看历史版本

文件被修改后，会自动保存历史版本：

1. 选中文件
2. 点击"..." → "版本"
3. 查看历史版本列表
4. 点击版本可预览

### 5.2 恢复旧版本

1. 在版本列表中
2. 找到需要的版本
3. 点击"恢复"按钮
4. 当前文件恢复为该版本

**注意:**
- 最多保留 10 个历史版本
- 超过 10 个会自动删除最旧的

## 七、手机 APP 使用技巧

### 7.1 离线访问

**设置离线文件:**
1. 长按文件/文件夹
2. 选择"设为可用离线"
3. 无网络时也可查看

### 7.2 照片自动上传

1. 打开 APP → 菜单 → 设置
2. 选择"自动上传"
3. 开启"照片和视频"
4. 选择上传目标文件夹
5. 选择"仅 WiFi 时上传"（省流量）

### 7.3 快速分享

1. 选中文件
2. 点击"分享"
3. 选择分享方式:
   - 微信
   - 邮件
   - 复制链接

## 八、常见问题

### Q1: 忘记密码怎么办？
A: 联系管理员重置密码

### Q2: 文件上传失败？
A: 检查：
- 文件大小是否超过限制（单个文件最大 10GB）
- 网络是否稳定
- 浏览器是否支持

### Q3: 在线编辑卡顿？
A: 尝试：
- 刷新页面
- 检查网络连接
- 关闭其他占用网络的程序

### Q4: 手机 APP 无法登录？
A: 检查：
- 服务器地址是否正确（https://开头）
- 是否连接公司 WiFi
- 账号密码是否正确

### Q5: 如何查看谁修改了文件？
A: 
- 文件 → 点击"..." → "活动"
- 可查看所有操作记录

### Q6: 误删文件怎么恢复？
A:
1. 进入"已删除文件"
2. 找到删除的文件
3. 点击"恢复"
4. 30天内可恢复

## 九、最佳实践

### 文件命名规范

- 客户项目：`年份-客户名-项目名称`
- 合同文件：`客户名-合同类型-日期`
- 现场照片：`日期-施工阶段-序号`
  - 例: `2025-01-15-水电-01.jpg`

### 文件夹管理

- 按项目分类存放
- 及时归档已完成项目
- 定期清理无用文件

### 安全提醒

- 不要将账号密码告诉他人
- 共享链接设置密码和有效期
- 离职人员及时通知管理员删除账号
